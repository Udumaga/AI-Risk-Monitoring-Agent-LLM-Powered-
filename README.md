# AI-Risk-Monitoring-Agent-LLM-Powered-

## Overview
This project demonstrates a lightweight AI agent that uses a Large Language Model (LLM) to detect, classify, and explain risk signals from unstructured text data.

The system simulates real-world AI governance workflows by transforming raw text into structured, actionable risk insights.

## Key Features
- Automated risk classification (security, bias, compliance, operational)
- Severity scoring (Low, Medium, High)
- Natural language explanation generation
- Reproducible and modular pipeline design

## Architecture
Input Text → LLM (OpenAI API) → Structured Risk Output → CSV 
## Tech Stack
- Python
- OpenAI API (LLM)
- Pandas
- Jupyter Notebook

## Example Output
| Text | Risk Type | Severity | Explanation |
|------|----------|---------|------------|
| Data exposure incident | Security | High | Sensitive data exposed due to misconfiguration |
